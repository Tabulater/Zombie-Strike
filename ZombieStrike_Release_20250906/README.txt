Zombie Strike - Installation Instructions
========================================

1. Install Python 3.8 or later from https://www.python.org/downloads/
   - During installation, make sure to check "Add Python to PATH"

2. Open Command Prompt and navigate to the game directory:
   cd path\to\ZombieStrike_Release_20250906

3. Install required packages:
   pip install -r requirements.txt

4. Run the game:
   python Tatipamula_Culminating.py

Controls:
- A/D or Left/Right Arrow: Move
- W or Up Arrow: Jump
- Space: Shoot
- R: Reload
- Left Click: Throw grenade (hold to aim)
- ESC: Pause/Quit to menu

For more details, see readme.md
