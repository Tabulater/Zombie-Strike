ZOMBIE STRIKE - INSTALLATION GUIDE
=================================


1. EXTRACT THE GAME FILES
   ---------------------
   - Right-click on the downloaded ZIP file
   - Select 'Extract All...'
   - Choose a destination folder (e.g., 'ZombieStrike' in your Documents)
   - Click 'Extract'

2. INSTALL PYTHON (if needed)
   --------------------------
   If you don't have Python 3.8 or later:
   - Visit https://www.python.org/downloads/
   - Click 'Download Python 3.x.x' (latest version)
   - RUN the installer
   - IMPORTANT: Check 'Add Python to PATH' at the bottom of the installer
   - Click 'Install Now'
   - Wait for installation to complete
   - Close and reopen any open command prompts

3. INSTALL REQUIRED PACKAGES
   ------------------------
- Run: pip install pygame

4. LAUNCH THE GAME
   --------------
   Option 1 - Easy Double-Click:
   - Simply double-click 'Launch Game.bat' in the game folder

OR 

- Run:  Tatipamula_Culminating.py

TROUBLESHOOTING
---------------
- If you get a 'python is not recognized' error:
  - Make sure Python is installed and added to PATH
  - Try using 'py' instead of 'python' in commands
  - Restart your computer after Python installation

- For other issues, please refer to README.md or contact support

ENJOY THE GAME!
--------------
- Use Arrow Keys to move and jump
- Space to shoot
- Left-click to throw grenades
- ESC to pause
